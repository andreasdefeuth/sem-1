HTML Formatting ^^

Cascading Style Sheets
Inline - by using the style attribute inside HTML elements.
Internal - by using a <\style> element in the <\head> section. Her skal man ikke bruge <\ og \> men i stedet brackets {\}. Style skal smides ind i head, og head skal lukkes før body introduceres
External - by using a <\link> element to link to an external CSS file. Her indsætter man <\link rel="stylesheet" href="styles.css"\> i head. Kan blive overwritten af Internal og Inline. Skrives med brackets som Internal. Det er også muligt at indsætte en URL
